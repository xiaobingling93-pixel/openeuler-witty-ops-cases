#标题
dnf update 时 mysql5-server 与 mariadb-server 软件包冲突

#内核版本
未知

#问题现象
执行 dnf update 时提示软件包冲突：mysql5-server-5.7.39-1.oe2203sp3.x86_64 与 mariadb-server（由 mariadb-server-1:10.5.22-1.oe2203sp3.x86_64 提供）冲突，导致无法安装 mariadb-server 及其相关更新包。

#问题根因
mysql5-server 与 mariadb-server 软件包存在互斥关系，不能同时安装。

#解决方案
- 在 dnf update 命令中添加 --allowerasing 参数以替换冲突的软件包；
- 或使用 --skip-broken 跳过无法安装的冲突包；
- 或使用 --nobest 以不强制使用最佳候选包的方式进行更新。
