#标题
PyCharm 连接远程 openEuler 无法准备 Python 环境

#内核版本
未知

#问题现象
PyCharm 连接远程 openEuler 服务器时提示“Failed to prepare environment”

#问题根因
SSH 服务未启用 TCP 转发功能，导致 PyCharm 无法通过 SSH 隧道建立远程环境连接

#解决方案
1. 编辑 /etc/ssh/sshd_config 文件，取消注释或添加 AllowTcpForwarding yes  
2. 执行 systemctl restart sshd 重启 SSH 服务  
3. 在 PyCharm 中清除缓存并重启
