#标题
cuda-samples测试用例因缺少X server显示环境执行失败

#内核版本
未知

#问题现象
执行`make test`编译并运行cuda-samples测试用例时，postProcessGL测试项报错“Waiving execution as X server is not running”，导致测试流程中断并返回错误码2。

#问题根因
测试用例postProcessGL依赖图形显示环境，但执行时系统未配置有效的X server显示目标。

#解决方案
1. 设置DISPLAY环境变量指向一个可远程显示的X server地址，例如：`export DISPLAY=128.224.32.151:0.0`
2. 重新执行测试命令，即可正常显示图像并通过测试。
