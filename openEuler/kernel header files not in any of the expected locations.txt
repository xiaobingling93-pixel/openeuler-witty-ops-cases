#标题
kernel header files not in any of the expected locations

#内核版本
未知

#问题现象
编译安装 Intel 驱动时，Makefile 报错：  
"*** Kernel header files not in any of the expected locations."  
"*** Install the appropriate kernel development package, e.g. kernel-devel, for building kernel modules and try again. Stop."

#问题根因
系统缺少 kernel-devel 包，或已安装但 /usr/src/linux 路径未指向实际的内核头文件目录。

#解决方案
1. 执行 `yum install kernel-devel` 安装 kernel-devel 包。  
2. 若已安装但仍报错，则在 /usr/src 目录下创建软链接：  
   `cd /usr/src`  
   `ln -s /kernels/xxxxxxxxxxx/ linux`（其中 xxxxxxxxxxx 为实际内核版本对应的目录）
