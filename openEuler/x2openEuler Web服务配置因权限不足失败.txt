#标题
x2openEuler Web服务配置因权限不足失败

#内核版本
未知

#问题现象
运行 sh service_init.sh 脚本后，提示“Failed to configure x2openEuler web service. Obtain the root permission or switch to the root user before performing this operation.”

#问题根因
执行 service_init.sh 脚本时未使用 root 权限，导致 x2openEuler web 服务配置失败。

#解决方案
1. 获取 root 权限或切换到 root 用户后再执行 sh service_init.sh 脚本。  
2. 升级至 8 月份发布的新版本以避免该问题。
