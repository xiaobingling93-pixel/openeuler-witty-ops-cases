#标题
升级过程中 install yum-utils failed 错误

#内核版本
未知

#问题现象
执行系统升级时持续报错：[ ERROR ] - [centos2openEuler]: install yum-utils failed, exit.；尝试卸载并重装 yum 相关软件包无效；本地 yum 源配置指向内网地址，但挂载目录为 /mnt 时系统提示需 umount，不挂载则报相同错误。

#问题根因
系统本地源挂载在 /mnt 目录导致升级脚本无法正常访问或处理 yum 源，从而引发 yum-utils 安装失败。

#解决方案
1. 将系统本地源挂载到非 /mnt 目录（例如其他自定义目录）；
2. 确保升级机使用 CentOS 源而非 openEuler 源进行升级操作。
