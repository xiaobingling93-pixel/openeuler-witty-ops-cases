#标题
x2openEuler Web登录密码忘记重置方法

#内核版本
未知

#问题现象
用户忘记 x2openEuler Web 页面的登录密码，无法正常登录系统。

#问题根因
x2openEuler 系统通过 is_firstlogin 字段控制用户是否为首次登录；重置该字段可触发密码重置流程。

#解决方案
1. 对于 2023 年 06 月 30 日之前部署的系统：
   - 进入目录 /usr/local/x2openEuler/portal/src/
   - 执行 sqlite3 db.sqlite3
   - 执行 SQL 命令：update usermanager_user set is_firstlogin=1;

2. 对于 2023 年 06 月 30 日之后部署的系统：
   - 执行 mysql -uroot -p 登录数据库
   - 使用数据库 x2openEulerDb：use x2openEulerDb;
   - 执行 SQL 命令：update usermanager_user set is_firstlogin=1;
