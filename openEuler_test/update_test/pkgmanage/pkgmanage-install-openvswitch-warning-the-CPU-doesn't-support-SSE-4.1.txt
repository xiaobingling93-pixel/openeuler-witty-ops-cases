# 标题
pkgmanage-install-openvswitch-warning-the-CPU-doesn't-support-SSE-4.1

# 基本信息
- 测试用例: oe_test_pkg_manager01
- 系统版本: openEuler-24.03-LTS-SP3
- 是否问题：否
- 关联issue：无

# 问题现象
pkgmanage测试执行oe_test_pkg_manager01用例过程中，安装软件包openvswitch执行安装脚本时候有提示Warning信息，用例捕捉到warning字段就提示用例执行失败。

# 错误日志

```
Warning: the CPU doesn't support SSE 4.1, dpdk support is not enabled.
```

# 问题根因

openvswitch软件包在spec中写了安装后执行%post脚本，如果有dpdk，就判断是否cpu支持SSE 4.1，不支持的话输出告警，属于正常告警信息

# 解决方案
1、确认openvswitch被成功安装（rpm -qa | grep openvswitch 或者 yum list installed | grep openvswitch）
2、成功被安装则无需解决
