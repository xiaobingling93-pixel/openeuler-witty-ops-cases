# 标题
pkgmanage “Packages upgrade error !”

# 基本信息
- 测试用例: oe_test_pkg_manager02
- 系统版本: openEuler-24.03-LTS-SP3(aarch64)

# 问题现象
pkgmanage测试执行过程中，检查软件包的升级，会检查升级过程中是否有报错信息出现。

# 错误日志
```
+ test -s /home/pkg_manager_folder/upgrade_fail_list
+ CHECK_RESULT 0 1 0 'Packages upgrade error !'
+ actual_result=0
+ expect_result=1
+ mode=0
+ error_log='Packages upgrade error !'
+ exit_mode=0
+ '[' -z 0 ']'
+ '[' 0 -eq 0 ']'
+ test 0x '!=' 1x
+ test -n 'Packages upgrade error !'
```

# 问题根因
1. 有冲突导致升级失败
2. 升级过程中会执行脚本，执行脚本有warnning，error，failed信息

# 解决方案
1. 排查/home/pkg_manager_folder/upgrade_fail_list，排查具体是哪个软件包升级失败，查看失败具体log
2. 从案例库中搜索对应软件包的案例来分析日志
