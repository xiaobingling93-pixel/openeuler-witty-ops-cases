# 标题
pkgcmd-dsctl-failed-cockpit_present is not defined

# 基本信息
- 测试用例: oe_test_dsctl_cockpit
- 系统版本: openEuler-24.03-LTS-SP1
- 是否问题：是
- 关联issue：https://atomgit.com/src-openeuler/three-eight-nine-ds-base/issues/46

# 问题现象
dsctl localhost cockpit open-firewall执行过程中报错

# 错误日志

Error: name 'cockpit_present' is not defined

# 问题根因

上游移除了函数cockpit_present的定义，但是还有地方在使用，导致出现报错Error: name 'cockpit_present' is not defined
上游社区移除方法的提交：[https://github.com/389ds/389-ds-base/pull/5615/commits/4d95d35aa3e687feccde79914f5aad06ea51b0cf]

# 解决方案
同步移除函数中调用到cockpit_present的代码
pr:https://atomgit.com/src-openeuler/three-eight-nine-ds-base/pull/98/
