# 问题根因
1、CHECK_RESULT 方法使用问题，最少需要一个参数，即获取上一条命令是否执行成功，而不是固定参数0。只传递参数0的话，校验结果恒成立，无意义。

# 解决方案
1、非问题，用例有误。修改用例：https://atomgit.com/openeuler/mugen/pull/5034
