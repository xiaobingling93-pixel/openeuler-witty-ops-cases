# 标题
pkgcmd “cpu型号为AMD family 25 CPU，mcelog不支持此型号”

# 基本信息
- 测试用例: oe_test_mcelog_02
- 系统版本: openEuler-24.03-LTS-SP3(x86_64)

# 问题现象
 mcelog --cpumhz 50命令执行失败

# 错误日志
```
mcelog: ERROR: AMD Processor family 25: mcelog does not support this processor.  Please use the edac_mce_amd module instead.
CPU is unsupported
```

# 问题根因
1、测试环境cpu型号为AMD family 25 CPU，mcelog不支持此型号。Family 25 = Zen4 架构（Ryzen 7000/8000、EPYC 9004 等），mcelog 是 Intel 为主的硬件错误日志工具，不支持新 AMD 架构。

# 解决方案
1、在cpu型号为Intel x86-64 处理器架构下执行oe_test_mcelog_02用例。
