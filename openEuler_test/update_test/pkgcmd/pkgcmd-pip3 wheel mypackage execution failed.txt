# 问题根因
1、从 Python 官方包仓库（files.pythonhosted.org）下载依赖包时，等待服务器响应的时间过长，导致连接中断。

# 解决方案
1、非问题，使用国内华为源下载需要的软件包，pip3 wheel mypackage -i https://repo.huaweicloud.com/repository/pypi/simple/  修改用例：https://atomgit.com/openeuler/mugen/pull/4959
