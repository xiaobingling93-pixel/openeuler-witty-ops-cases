# 标题
pkgcmd “aarch64架构下 secscanner check cve_t 执行失败，x86架构执行通过”

# 基本信息
- 测试用例: oe_test_secscanner_004
- 系统版本: openEuler-24.03-LTS-SP3(aarch64)
- 是否问题: 否
- 关联issue: 无

# 问题现象
aarch64架构下 secscanner check cve_t 执行失败，x86架构执行通过。

# 问题根因
1、This arm architecture is not supported by the vulnerability scanning feature at this time.  用例不健全，该测试用例漏洞扫描未支持arm架构。

# 解决方案
1、非问题，单架构测试用例，以x86执行结果为准。 修改用例，https://atomgit.com/openeuler/mugen/pull/4854
