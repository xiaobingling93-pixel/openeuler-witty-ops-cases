# 标题
pkgserver “samba服务报端口冲突的错误”

# 基本信息
- 测试用例: oe_test_service_samba
- 系统版本: openEuler-24.03-LTS-SP3(aarch64/x86_64)
- 是否问题: 否
- 关联issue: https://gitee.com/src-openeuler/samba/issues/I6VTA2?from=project-issue

# 问题现象
测试执行过程中，检查到服务执行过程中报错绑定TCP 53端口失败

# 问题根因
1. 53端口默认由DNS服务（如named、dnsmasq）占用。Samba尝试绑定TCP 53端口失败。

# 解决方案
1. 检查53端口占用
netstat -tulnp | grep :53
2.杀死占用53端口的进程（假设PID为1234）
sudo kill -9 1234
