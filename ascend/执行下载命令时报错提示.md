# 执行下载命令时报错提示

## 内核版本


## 问题现象
使用Ascend Deployer工具执行下载命令时，报错提示“download failed with error url open failed, please check the network ,please retry”。

## 问题根因
网络连接异常、网络不稳定或服务器异常断连，也可能是配置的PyPI源不可用。

## 解决方案
首先检查当前环境的网络连接是否正常；若网络正常，可尝试重新执行下载命令；如果问题仍然存在，修改ascend-deployer/ascend_deployer/downloader/config.ini文件中的pypi源为其他可用源后重试。
