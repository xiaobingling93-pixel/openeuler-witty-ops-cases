# Atlas 800-3010 Windows2012R2系统, 不登录系统，通过BMC WEB进行安全重启，单板直接下电然后重启，没有看到关机过程

## 内核版本


## 问题现象
在SBCA单板上安装Windows Server 2012 R2系统后，若未登录系统，通过BMC Web界面执行安全重启操作时，单板直接断电并重启，未观察到正常的关机（shutdown）过程。

## 问题根因
在系统上电但用户未登录前，Windows的电源管理服务尚未启动。因此，BMC发出的安全重启指令无法触发系统的正常关机流程。只有在用户登录后，电源管理服务启动，系统才能响应BMC的安全重启命令并执行关机过程。此行为是Windows Server 2012 R2自身的电源管理策略所致。

## 解决方案
该问题属于Windows Server 2012 R2系统默认行为，不影响用户正常使用和业务运行，因此无需解决。
