# 模型转换失败，报错日志中有“can not find node: *”或者“node * not found”的内容信息

## 内核版本


## 问题现象
模型转换失败，报错日志中出现“can not find node: *”或“node * not found”的错误信息。

## 问题根因
1. 使用out_nodes参数指定的输出节点（算子名称）在模型图（Graph）中不存在；2. 指定的输出节点虽然存在于原始图中，但在模型转换过程中被融合优化掉，导致最终图中无法找到该节点。

## 解决方案
1. 检查模型图中是否存在out_nodes参数指定的输出节点，若不存在则需更正为正确的节点名称；2. 若节点在原始图中存在但仍报错，可尝试不带out_nodes参数进行模型转换，然后检查最终生成的图中是否包含该节点，若已被融合，则需重新选择未被融合的节点作为输出节点。
