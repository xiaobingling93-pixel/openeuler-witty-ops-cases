# 引用库找不到的问题

## 内核版本


## 问题现象
在使用MindStudio过程中，出现引用库找不到的问题。

## 问题根因
可能的原因包括：1）引用的库位于当前目录或子目录中，但其父目录未添加到PYTHONPATH环境变量；2）所需的库在requirements.txt中声明，但未通过pip正确安装；3）某些库需通过readme.md中说明的方式（如git clone）手动下载安装，但用户未执行相应操作。

## 解决方案
1. 若引用库位于当前目录或其子目录中，将该目录的父目录添加到PYTHONPATH环境变量；2. 若库在requirements.txt中列出，使用命令 pip install xxx -i http://cmc-cd-mirror.rnd.huawei.com/pypi/simple/ --trusted-host cmc-cd-mirror.rnd.huawei.com 安装，若失败可尝试 git clone 后通过 python3 setup.py install 安装；3. 检查readme.md是否要求通过git clone方式安装特定库，并按说明操作。
