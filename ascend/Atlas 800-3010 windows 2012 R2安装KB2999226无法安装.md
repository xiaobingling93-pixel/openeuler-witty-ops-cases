# Atlas 800-3010 windows 2012 R2安装KB2999226无法安装

## 内核版本
Windows 2012 R2 (内核版本 9600.16384)

## 问题现象
在Atlas 800-3010服务器上安装Windows Server 2012 R2操作系统后，尝试安装更新补丁KB2999226时失败，无法完成安装。

## 问题根因
KB2999226补丁与特定版本的Windows Server 2012 R2内核存在兼容性问题。具体而言，使用内核版本为9600.16384.WINBLUE_RTM.130821-1623的系统镜像时无法安装该补丁，而使用更新的镜像版本9600.17050.WINBLUE_REFRESH.140317-1640则可以正常安装，说明问题根源在于操作系统内核版本与KB补丁之间的适配性。

## 解决方案
建议使用Windows Server 2012 R2内核版本为9600.17050或更高版本的系统镜像进行业务部署，以确保KB2999226补丁能够正常安装。
