# 量化模型时校准集数据大小与模型输入大小不匹配，量化模型失败

## 内核版本


## 问题现象
执行ATC模型转换命令进行模型量化时，报错提示“input image size[1579014] is not equal to model input size[3158028]”，导致量化失败。

## 问题根因
校准集数据的shape或数据类型与模型输入的shape或数据类型不一致，导致输入数据大小与模型期望的输入大小不匹配。

## 解决方案
逐一排查校准集数据的shape和数据类型是否与模型输入的shape和数据类型一致。模型输入shape通过量化配置文件中的input_shape参数配置，模型输入数据类型需从模型来源获取或通过第三方工具查看模型文件确认。
