# 加载集群场景性能数据时提示“Low memory”

## 内核版本


## 问题现象
MindStudio在加载集群场景下采集的性能原始数据时，提示“Low memory”错误。

## 问题根因
集群场景下采集的原始性能数据量较大，加载到内存中需要占用较多内存空间，而MindStudio默认分配的虚拟机内存不足，导致加载过程中出现内存不足的异常提示。

## 解决方案
通过调整MindStudio的虚拟机内存配置解决：1. 点击顶部菜单栏“Help > Edit Custom VM Options...”；2. 将虚拟机内存大小调整为原内存的5-10倍（例如从2048MB调整为20480MB）；3. 保存配置并重启MindStudio。
