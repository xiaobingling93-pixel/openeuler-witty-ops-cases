# 内存诊断采集时报错的解决方法

## 内核版本
4.19.90-vhulk2211.3.0.h1489.eulerosv2r10.aarch64

## 问题现象
内存诊断采集时报错，报错信息：modprobe:FATAL:Module kheaders not found in directory /lib/modules/4.19.90-vhulk2211.3.0.h1489.eulerosv2r10.aarch64

## 问题根因
/lib/modules下对应的内核版本目录中不包含build文件，无法查询到对应的kernel-headers。

## 解决方案
1. 安装依赖：执行命令 yum install kernel-devel；2. 查看软链接：在/lib/modules下进入对应的aarch64版本目录（yum安装的kernel-devel版本），查看build对应的软链接；3. 添加软链接：在/lib/modules下对应内核版本目录中添加build的软链接，可先通过uname -a查看对应内核版本，然后执行命令 ln -sf 目标文件 build。
