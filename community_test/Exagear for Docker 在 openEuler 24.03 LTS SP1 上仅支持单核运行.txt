#标题  
Exagear for Docker 在 openEuler 24.03 LTS SP1 上仅支持单核运行

#内核版本  
24.03sp1

#问题现象  
Exagear for Docker 在 openEuler 22.03 LTS 系列版本中可正常多核运行应用，但在升级至 openEuler 24.03 LTS SP1 后，应用仅能以单核模式运行，无法利用多核资源。

#问题根因  
openEuler 24.03 LTS SP1 内核或相关运行时环境变更导致 Exagear for Docker 的多核调度机制失效，具体表现为兼容性问题。

#解决方案  
建议排查 openEuler 24.03 LTS SP1 中与 CPU 调度、cgroups 或容器运行时相关的内核配置或安全策略变更；联系 Exagear 厂商确认其对 24.03 LTS SP1 的兼容性支持，并获取适配该版本的更新补丁或配置指南。
